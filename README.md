# Welcome to My Portfolio Website  

## Background  

I developed this personal website to demonstrate my technical proficiency in web management and version control. By utilizing the command line interface and Git, I have built a platform to showcase my qualifications and projects for my professional career.

## Takeaways  

During the development process, I successfully navigated several technical challenges, including:  

* **GitHub Management**: Creating repositories, generating access tokens, forking, and managing version control.  
* **Environment Setup**: Configuring Jekyll and resolving Ruby version mismatches within the macOS environment.  
* **Git Proficiency**: Mastering essential Git commands for efficient workflow.  
* **Network Configuration**: Managing DNS settings and URL redirection via Namecheap.  

As anticipated, troubleshooting within the CLI required the most persistence. Successfully deploying this site has not only provided me with a functional portfolio but also strengthened my foundational skills in systems and network management.
