## My Windows history
My journey began in 1995 at the age of five with MS-DOS. Because my father owned a computer shop, I learned how to assemble hardware and install software even before I started school. Since then, I have moved through Windows 95, 98, 2000, ME, XP, 7, 8, 10, and currently use Windows 11 alongside macOS and a Synology Linux server. Windows XP remains my personal favorite. It likely holds a special place in my heart because it reminds me of my happiest and most carefree years as a student.

## Legendary Windows Games
I still vividly remember my first MS-DOS games: a card-flipping memory game, a skiing game where you flee from a chasing figure, and a game where a cat chases your mouse pointer. Later came Minesweeper, and Windows XP featured the masterpiece known as 3D Pinball. Microsoft eventually stopped including Pinball because the original developers did not document how the code worked. This made it nearly impossible for later engineers to fix bugs or update the game for newer systems.

## OS Installation Challenges of the Past
The difficulty of installing an operating system changed significantly between the eras of Windows XP and Windows 7. In the old days, you had to insert a boot disk and repeatedly tap the DEL key to enter the BIOS. Once there, you had to manually set the boot priority. Because we did not have SSDs back then, I sometimes started an installation and went to sleep while it processed. Even after the OS was finished, the work was not over. You had to manually install every hardware driver. If you missed the graphics driver, the screen would default to a low 800x600 resolution, making the icons look huge and the computer appear quite primitive.

## Software Installation Hurdles
Installing software back then wasn't as simple as a few clicks. Since my father ran a computer shop, our house was always overflowing with Windows CDs and serial numbers. However, many of my friends lived in rural areas where software was scarce. I even heard stories of people traveling all the way to Daegu or Seoul just to buy MS Office CDs. On the other hand, it was common to borrow original discs for games like StarCraft or Diablo from older neighbors, "burn" them onto blank CDs using Nero and a CD-RW drive, and then use virtual drive software like Daemon Tools or CD Space to play them. This is something you no longer have to do these days. Korea has since become one of the wealthiest countries in the world, and stores are now everywhere.
